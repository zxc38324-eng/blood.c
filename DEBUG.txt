📝 ЧАСТЫЕ ОШИБКИ:

1. "This game is not supported"
   → Ты не в Boom Hood (PlaceId: 120822170036689)
   → Решение: Зайди в правильную игру

2. "404: Not Found"
   → Файлы не на GitHub
   → Решение: Загрузи папку blood_modular на GitHub

3. "HttpService is not allowed"
   → Эксплойт блокирует HTTP запросы
   → Решение: Используй другой эксплойт

4. "loadstring is not available"
   → Эксплойт не поддерживает loadstring
   → Решение: Используй другой эксплойт

5. Скрипт загружается но ничего не происходит
   → UI библиотека не загрузилась
   → Решение: Проверь интернет, попробуй еще раз
